# 9. Palindrome Number

## 题目
判断一个整数是否是回文数（正着读和反着读一样）。

## 思路

### 方法1：转字符串
将数字转为字符串，比较是否等于其反转。

```python
class Solution:
    def isPalindrome(self, x: int) -> bool:
        if x < 0:
            return False
        return str(x) == str(x)[::-1]
```

- **时间复杂度：O(n)**，n 为数字位数
- **空间复杂度：O(n)**，需额外存储字符串和反转字符串

### 方法2：反转后半段（最优）
不用字符串，只反转数字的后半段，再与前半段比较。

```python
class Solution:
    def isPalindrome(self, x: int) -> bool:
        if x < 0 or (x % 10 == 0 and x != 0):
            return False
        
        reversed_half = 0
        while x > reversed_half:
            reversed_half = reversed_half * 10 + x % 10
            x //= 10
        
        return x == reversed_half or x == reversed_half // 10
```

- **时间复杂度：O(n/2)**，只遍历一半位数
- **空间复杂度：O(1)**，只用常数个变量

## 关键点
- 负数直接返回 `False`
- 末位是 `0` 但本身不是 `0` 的数（如 10、100）直接返回 `False`
- 奇数位时（如 121），停止条件 `x == reversed_half // 10` 处理中间那位
- Python 布尔值注意大小写：`False` 而非 `false`
