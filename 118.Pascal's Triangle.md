# 118. Pascal's Triangle

- **难度**: Easy
- **链接**: https://leetcode.com/problems/pascals-triangle/

## 思维误区复盘（按踩坑顺序）

### 1. 把 `numRows`（行数）当成了数组来用

**❌ 错误写法**

```python
def generate(self, numRows: int) -> List[List[int]]:
    n = len(numRows)          # numRows 是 int，不能 len()
    for i in range(n):
        for j in range(n):
            result[j] = ...
```

**✅ 正确写法**

```python
def generate(self, numRows: int) -> List[List[int]]:
    ans = []
    for i in range(numRows):  # numRows 本身就是循环次数
        ...
```

### 2. 返回类型搞混，多此一举地 join 成字符串

**❌ 错误写法**

```python
ans = result.join(",")   # 返回类型是 List[List[int]]，根本不需要拼成字符串
return ans
```

**✅ 正确写法**

```python
ans = []
# ... 把每一行 list 直接 append 进 ans
return ans                # 直接返回二维列表
```

### 3. 内层循环范围越界，没有跳过边界元素（首尾恒为1）

**❌ 错误写法**

```python
row = [1] * (i + 1)
for j in range(i):                       # j=0 也被包含，多算了
    row[j] = ans[i-1][j-1] + ans[i-1][j]  # i=0 时 ans[-1] 越界；j=0 时取错了上一行的元素
```

**✅ 正确写法**

```python
row = [1] * (i + 1)
for j in range(1, i):                     # 跳过 j=0 和 j=i，这两个位置已经是1
    row[j] = ans[i-1][j-1] + ans[i-1][j]
```

## 最终解法

```python
from typing import List

class Solution:
    def generate(self, numRows: int) -> List[List[int]]:
        ans = []
        for i in range(numRows):
            row = [1] * (i + 1)
            for j in range(1, i):
                row[j] = ans[i-1][j-1] + ans[i-1][j]
            ans.append(row)
        return ans
```

- **时间复杂度**: O(numRows^2)
- **空间复杂度**: O(numRows^2)
