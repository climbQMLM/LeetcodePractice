# 94. Binary Tree Inorder Traversal

- **难度**: Easy
- **链接**: https://leetcode.com/problems/binary-tree-inorder-traversal/

## 知识点 Q&A

**Q: `dfs` 是定义在类方法内部的嵌套函数，为什么它的入参不用写 `self`？**

A: `self` 之所以要写，是因为类的方法被调用时（如 `obj.inorderTraversal(root)`）会自动把实例本身作为第一个参数传进去，所以方法定义里要显式接收它。`dfs` 不是通过 `self.dfs(...)` 这种方式调用的，而是在 `inorderTraversal` 内部直接像普通函数一样调用 `dfs(node)`，没有实例绑定的过程，所以不需要 `self`。可以把 `dfs` 理解成在 `inorderTraversal` 函数体里临时定义的一个普通函数，只是借助闭包能访问外层变量（比如 `ans`）。

**Q: `Optional[TreeNode]` 是什么意思？**

A: 这是类型注解，表示这个值的类型是 `TreeNode` 或者 `None`，等价于 `Union[TreeNode, None]`。在二叉树问题里，递归到叶子节点的子节点时 `node.left`/`node.right` 会是 `None`，所以用 `Optional[TreeNode]` 准确描述这种"可能为空"的情况，这只是类型提示，不影响代码实际运行。

## 思维误区复盘（按踩坑顺序）

### 1. 用方括号调用函数，与索引操作混淆

**❌ 错误写法**

```python
def dfs(node):
    if node is None:
        return
    dfs[node.left]    # [] 是索引/下标操作，不是函数调用
    dfs[node.right]
```

**✅ 正确写法**

```python
def dfs(node):
    if node is None:
        return
    dfs(node.left)     # 函数调用要用圆括号
    dfs(node.right)
```

### 2. 只递归了左右子树，漏了"访问当前节点"这一步（中序遍历的核心）

**❌ 错误写法**

```python
def dfs(node):
    if node is None:
        return
    dfs(node.left)
    dfs(node.right)    # 缺少处理当前节点的逻辑，没有体现"中序"
```

**✅ 正确写法**

```python
def dfs(node):
    if node is None:
        return
    dfs(node.left)       # 左
    ans.append(node.val) # 根
    dfs(node.right)      # 右
```

### 3. append 进去的是节点对象，而不是节点的值

**❌ 错误写法**

```python
ans.append(node)      # 添加的是 TreeNode 对象，但返回类型要求 List[int]
```

**✅ 正确写法**

```python
ans.append(node.val)  # 添加节点的值
```

## 最终解法

```python
from typing import Optional, List

class Solution:
    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        ans = []
        def dfs(node: Optional[TreeNode]):
            if node is None:
                return
            dfs(node.left)
            ans.append(node.val)
            dfs(node.right)
        dfs(root)
        return ans
```

- **时间复杂度**: O(n)
- **空间复杂度**: O(n)（递归栈，最坏情况树退化成链表）
