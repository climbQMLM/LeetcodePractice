# 94. Binary Tree Inorder Traversal

- **难度**: Easy
- **链接**: https://leetcode.com/problems/binary-tree-inorder-traversal/

## 思维误区复盘（按踩坑顺序）

### 1. 用方括号调用函数，与索引操作混淆

**❌ 错误写法**

```python
def dfs(node):
    if node is None:
        return
    dfs[node.left]    # [] 是索引/下标操作，不是函数调用
    dfs[node.right]
```

**✅ 正确写法**

```python
def dfs(node):
    if node is None:
        return
    dfs(node.left)     # 函数调用要用圆括号
    dfs(node.right)
```

### 2. 只递归了左右子树，漏了"访问当前节点"这一步（中序遍历的核心）

**❌ 错误写法**

```python
def dfs(node):
    if node is None:
        return
    dfs(node.left)
    dfs(node.right)    # 缺少处理当前节点的逻辑，没有体现"中序"
```

**✅ 正确写法**

```python
def dfs(node):
    if node is None:
        return
    dfs(node.left)       # 左
    ans.append(node.val) # 根
    dfs(node.right)      # 右
```

### 3. append 进去的是节点对象，而不是节点的值

**❌ 错误写法**

```python
ans.append(node)      # 添加的是 TreeNode 对象，但返回类型要求 List[int]
```

**✅ 正确写法**

```python
ans.append(node.val)  # 添加节点的值
```

## 最终解法

```python
from typing import Optional, List

class Solution:
    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        ans = []
        def dfs(node: Optional[TreeNode]):
            if node is None:
                return
            dfs(node.left)
            ans.append(node.val)
            dfs(node.right)
        dfs(root)
        return ans
```

- **时间复杂度**: O(n)
- **空间复杂度**: O(n)（递归栈，最坏情况树退化成链表）
