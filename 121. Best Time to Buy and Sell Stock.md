# 121. Best Time to Buy and Sell Stock

- **难度**: Easy
- **链接**: https://leetcode.com/problems/best-time-to-buy-and-sell-stock/

## 思路推导

给一个价格数组，只能买卖一次，求最大利润。

关键观察：利润 = 某天卖出价 - 之前某天的最低买入价。只需一次遍历，每天假设"今天卖出"，买入点取历史最低价，计算当天利润并更新最大值。

这是贪心思路：无需枚举所有买卖组合，只要维护两个变量即可。

## 关键点

- `min_price` 初始化为 `float('inf')`，因为 Python 的 `int` 没有正无穷，`float('inf')` 保证第一个价格一定会更新它
- 等价写法是 `min_price = prices[0]`，两种都对
- 先更新 `min_price`，再算利润（顺序不能反，否则可能用当天价格既买又卖）

## 最终解法

```python
def maxProfit(prices):
    min_price = float('inf')
    max_profit = 0
    for price in prices:
        min_price = min(min_price, price)
        max_profit = max(max_profit, price - min_price)
    return max_profit
```

- **时间复杂度**: O(n)
- **空间复杂度**: O(1)
