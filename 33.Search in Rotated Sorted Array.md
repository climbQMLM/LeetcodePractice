# 33. Search in Rotated Sorted Array

- **难度**: Medium
- **链接**: https://leetcode.com/problems/search-in-rotated-sorted-array/

## 思维误区复盘（按踩坑顺序）

### 1. 用 `=` 代替 `==` 做判断，且单元素特判其实多余

**❌ 错误写法**

```python
n = len(nums)
if n = 1 and nums[0] != target:   # = 是赋值，会语法报错；且这个特判没必要
    return -1
```

**✅ 正确写法**

```python
n = len(nums)
left, right = 0, n - 1
# 不需要单独特判 n==1，主循环本身就能处理
```

### 2. mid 计算少了 `//2`，且要在循环内重新计算

**❌ 错误写法**

```python
while left <= right:
    mid = left + (right - left)   # 少了 //2，mid 直接等于 right
    if nums[mid] == target:
        ...
```

**✅ 正确写法**

```python
while left <= right:
    mid = left + (right - left) // 2   # 每次循环都要重新计算 mid
    if nums[mid] == target:
        return mid
```

### 3. 判断 mid 落在哪个有序区间时，条件写反/不完整

**❌ 错误写法**

```python
if nums[mid] > nums[left]:   # 没处理 mid 落在右半段有序的情况
    if nums[left] <= target <= nums[mid]:
        right = mid - 1
    else:
        left = mid + 1
else:
    # 卡住，不知道这里该怎么判断
    pass
```

**✅ 正确写法**

```python
if nums[mid] > nums[left]:        # 左半段 [left, mid] 有序递增
    if nums[left] <= target <= nums[mid]:
        right = mid - 1
    else:
        left = mid + 1
else:                              # 右半段 [mid, right] 有序递增
    if nums[mid] <= target <= nums[right]:
        left = mid + 1
    else:
        right = mid - 1
```

## 最终解法

```python
from typing import List

class Solution:
    def search(self, nums: List[int], target: int) -> int:
        n = len(nums)
        left, right = 0, n - 1
        while left <= right:
            mid = left + (right - left) // 2
            if nums[mid] == target:
                return mid
            if nums[mid] > nums[left]:        # 左半段有序
                if nums[left] <= target <= nums[mid]:
                    right = mid - 1
                else:
                    left = mid + 1
            else:                              # 右半段有序
                if nums[mid] <= target <= nums[right]:
                    left = mid + 1
                else:
                    right = mid - 1
        return -1
```

- **时间复杂度**: O(log n)
- **空间复杂度**: O(1)
